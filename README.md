## Описание проекта

**TaskBoard** — это полнофункциональное веб-приложение для управления задачами (Todo Board), разработанное с использованием современных технологий Fullstack.

### Возможности приложения:
- Просмотр списка всех задач
- Добавление новых задач (с названием и описанием)
- Отметка задач как выполненных (с визуальным перечёркиванием)
- Редактирование задач прямо в карточке (inline-редактирование)
- Удаление задач с подтверждением
- Фильтрация задач: Все / В работе / Выполненные
- Счётчик выполненных задач
- Адаптивный и современный дизайн

---

## Технологический стек

**Backend:**
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- Swagger (для тестирования API)

**Frontend:**
- HTML5, CSS3 (Flexbox)
- Vanilla JavaScript (Fetch API)
- Семантическая вёрстка

## Главные выводы

1. **Полный цикл Fullstack-разработки**: Я самостоятельно построил приложение от создания базы данных до интерактивного пользовательского интерфейса, пройдя весь путь от backend до frontend.

2. **Работа с REST API**: Реализовал все основные HTTP-методы (GET, POST, PUT, DELETE) на стороне сервера и научился правильно вызывать их с фронтенда с помощью Fetch API.

3. **Entity Framework Core + SQLite**: Освоил работу с ORM, создание модели, контекста базы данных, seeding начальных данных и автоматическое применение миграций.

4. **Клиент-серверное взаимодействие**: Научился обрабатывать асинхронные запросы, корректно работать с JSON, обрабатывать ошибки и обновлять интерфейс без перезагрузки страницы.

5. **Современные практики frontend**: Применил модульный подход (отдельные файлы `api.js` и `app.js`), использовал шаблонные строки, динамическое создание HTML, управление состоянием приложения и защиту от XSS через `escapeHtml()`.

6. **Inline-редактирование и фильтрация**: Реализовал удобный UX с переключением режимов просмотра/редактирования и клиентской фильтрацией задач без дополнительных запросов к серверу.
